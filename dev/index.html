Outline example

A few simple examples of including an outline component.

Files needed

Include the outline.js and outline-legacy.js files.

Include a copy of outline.theme.css.

The CSS file includes many CSS variables that you can customize for your project.

Outline components will use these CSS variables in their rules. This stylesheet applies all the variables to the :root selector which in most cases is the document. 

      <link rel="stylesheet" href="outline.theme.css">
      <script type="module" src="outline.js"></script>
      <script nomodule src="outline-legacy.js"></script>
