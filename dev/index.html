How this environment is used

This environment is powered by Web Dev Server (wds) and the final production output located in the dist directory. This allows use the ability to preview component content in a production consumer environment without ever leaving the frontend development system. Storybook provides our development presentation, but can sometimes present content or renderings slightly differently. This helps us test early, often, and thoroughly.
