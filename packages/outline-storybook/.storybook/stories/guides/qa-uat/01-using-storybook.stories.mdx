## Introduction to Storybook

### Storybook Basics
- Use the search feature to find the component or page you are looking for.
- Use the controls to manipulate various components in.
- Note the `Canvas` and `Docs` tabs at the top of some components. 
  - `Docs` are a bit more friendly to view/read.
  - `Canvas` offers more tools, including accessiblity plugin among others. 
- Note the Prototypes section in the sidebar for full page samples.
